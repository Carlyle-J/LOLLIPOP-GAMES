# **LOLLIPOP GAMES – Game E-Commerce Website**

Welcome to **LOLLIPOP GAMES**, a fully-designed gaming e-commerce website created using **HTML, CSS, Bootstrap, and JavaScript**.
This project showcases a complete shopping experience with product listings, category browsing, and an admin dashboard for managing products.

---

## **Project Overview**

**LOLLIPOP GAMES** is a responsive web platform for browsing and purchasing video games and gaming accessories.
The website features a **gold, yellow, black and neon theme**.
The site includes both **user-facing pages** and **admin-only pages** for managing items on the store.

---

## **Main Features**

### **User Features**

* Browse by gaming category:

  *  Accessories
  *  PlayStation Games
  *  Nintendo Games
  *  Xbox Games
  *  Console Games
* Modern UI with a neon–gold theme
* Responsive layout built with **Bootstrap**
